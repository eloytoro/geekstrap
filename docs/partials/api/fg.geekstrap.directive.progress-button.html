progress-button
directive in module fg.geekstrap

Description

A button that morphs into a loading bar whenever it's model value changes

Usage

as element:
<progress-button
       [model="{number}"]
       [onload="{function}"]>
</progress-button>

Parameters

ParamTypeDetails
model
(optional)
number

value within the scope that determines progress (from 0 to 100)

onload
(optional)
function

progress completion callback
